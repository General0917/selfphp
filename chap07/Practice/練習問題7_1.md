# 練習問題7.1

## クラスと関数、メソッドの違いを以下の表に記載する

|    |  データを保持  |  データを処理  |
| ---- | ---- | ---- |
|  変数  |  〇  |  ×  |
|  関数  |  ×  |  〇  |
|  クラス/オブジェクト  |  〇  |  〇  |

## クラスを利用する際には、クラスをコピーしてオブジェクトを作成する必要があることは何か？

答え：インスタンス化
<br>
クラスを利用するにあたっては、インスタンス化は必須の手続きである、
<br>
インスタンス化には「new」演算子を用いる。
<br>
例：$変数名 = new クラス名([引数, ‥‥‥])
